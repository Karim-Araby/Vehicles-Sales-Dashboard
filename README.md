Project Overview

An interactive Vehicles Sales Dashboard built to analyze sales performance across products, countries, regions, and years. The dashboard highlights key metrics such as total sales, best-selling product categories, and regional distribution, helping users quickly identify trends and business insights.

This project presents an interactive Vehicles Sales Dashboard created in Microsoft Excel to analyze sales performance across different products, countries, regions, and years.

The dashboard provides a clear overview of key metrics and helps identify trends and patterns in vehicle sales.

# Tools Used

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers

Data Visualization

# Key Insights

Overview of total vehicle sales

Identification of the best-selling product category

Sales comparison across regions and countries

Monthly sales trend analysis

# Project Purpose

The goal of this project is to demonstrate how Excel dashboards can be used to transform raw data into meaningful insights through interactive visualizations.
